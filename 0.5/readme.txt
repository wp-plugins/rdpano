Intégration de panorama krpano, panotour® et panoStudio sur WordPress

== Description ==

RDPano est un plugin pour intégrer facilement vos panoramas krpano, panotour® et panoStudio sur la plateforme WordPress.

L'intégration de vos panoramas devient un jeu d'enfant et votre animation 360° est vue par tous !

== Installation ==

1. Charger le dossier "rdpano" dans le dossier "/wp-content/plugins"
2. Activer l'extension (=plugin) RDPano via le menu "Extensions" dans WordPress
